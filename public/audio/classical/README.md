# Classical Music Audio Files

These audio files are public domain recordings from Wikimedia Commons, used for the Classical Music Quiz game.

## Source
All audio files were sourced from [Wikimedia Commons](https://commons.wikimedia.org/), a repository of freely-usable media files.

## License
The recordings are in the **public domain** or licensed under Creative Commons licenses that permit redistribution. The musical compositions themselves are public domain (composers deceased > 70 years).

## Contents
50 classical music pieces from various composers including:
- Ludwig van Beethoven
- Wolfgang Amadeus Mozart  
- Johann Sebastian Bach
- Pyotr Ilyich Tchaikovsky
- Frédéric Chopin
- Claude Debussy
- And many more

## File Format
Files are in MP3 or OGG format for browser compatibility.
