## A.K.A.

Also known as: Sequence Memory, Number Memory Test, Working Memory Test

## Category

Memory Games

## How to Play

1. Numbers appear briefly in randomized positions on a grid
2. After a short display period, the numbers are hidden
3. Click the boxes in ascending order (1, 2, 3...)
4. Each successful round adds more numbers to remember
5. Game ends when you click the wrong box

## Difficulty Levels

- **Easy**: Start with 3 numbers
- **Medium**: Start with 4 numbers
- **Hard**: Start with 5 numbers

## Tips & Strategy

- Focus on spatial patterns rather than memorizing individual positions
- Try grouping numbers into chunks (like phone numbers)
- Take a mental "snapshot" of the layout during the display phase
- Practice regularly to improve your working memory capacity
