# üß† Memorize Items

Test your visual memory by remembering which items were shown.

## A.K.A.

Also known as: Item Recall, Visual Memory Test, Memory Challenge

## Category

Memory Games

## How to Play

1. A set of items (emojis) is displayed briefly
2. After the time expires, additional distractor items appear
3. Select all the items you remember seeing in the original set
4. Submit your selections to score and proceed to the next round
5. Rounds continue indefinitely, tracking your cumulative score

## Difficulty Levels

- **Easy**: 6 items to memorize, 5 seconds display time
- **Medium**: 8 items to memorize, 4 seconds display time
- **Hard**: 10 items to memorize, 3 seconds display time

## Scoring

- Points awarded for each correctly identified item
- Track your best score per difficulty level

## Item Categories

Items include fruits, animals, technology, vehicles, sports equipment, and more.

## Puzzle Source

**üîß Generator**

**Source:** Procedural generation with daily seed

## Tips & Strategy

- Group items by category to aid recall (e.g., "three animals, two fruits")
- Create mental stories connecting the items
- Focus on distinctive items first, then fill in common ones
- Use spatial memory - try to remember where items appeared
- Regular practice improves working memory capacity
