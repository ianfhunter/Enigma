Find groups of 3 cards where each attribute is either all the same or all different!

## A.K.A.

Also known as: Set Game, Set Card Game

## Category

Classic Logic & Reasoning

## How to Play

1. Each card has 4 attributes: color, shape, fill, and count
2. Find groups of 3 cards where each attribute is either:
   - All the same across the 3 cards, OR
   - All different across the 3 cards
3. Click 3 cards to form a Set
4. Valid Sets are highlighted and removed from play
5. Find all Sets in the puzzle to win

## Tips & Strategy

- Look for obvious patterns first (3 cards with same shape, all different colors)
- Remember: EACH attribute must be all-same or all-different
- Start with attributes that are easiest for you to see (often color or shape)
- Use the Hint button to reveal one card from a valid Set
